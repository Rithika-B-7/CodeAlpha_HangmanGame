# 🕹️ Hangman Game - Python

> Developed as part of CodeAlpha Python Programming Internship

## 📌 Overview

A simple **text-based Hangman Game** built using Python. The player guesses a randomly selected word one letter at a time within limited attempts.

---

## 🚀 Features

* 🎮 Interactive command-line gameplay
* 🔤 Random word selection
* ✅ Input validation
* 📉 6 incorrect attempts limit
* 📋 Displays guessed letters & progress
* 🔁 Replay option

---

## 🛠️ Tech Used

* Python
* Random module

---

## ▶️ How to Run

```bash
git clone https://github.com/Rithika-B-7/CodeAlpha_HangmanGame.git
cd CodeAlpha_HangmanGame
python3 hangman.py
```
